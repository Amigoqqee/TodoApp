# TodoApp

Простое приложение для ведения списка задач.

## Функционал
- Добавление, отметка выполнения и удаление задач.
- Сохранение списка задач в базе данных.

## Технологии
- Backend: Go (Golang)
- База данных: MongoDB Atlas

## Запуск
1. Клонировать репозиторий.
2. Установить зависимости и переменные окружения для доступа к MongoDB Atlas.
3. Запустить сервер: `go run ./cmd/server` (или своя команда).
